# Code Book

This code book summarizes the corresponding variables in `tidy_data.txt`.

## Identifiers

* `subject` - The ID of the test subject
* `activity` - The type of activity performed when the corresponding measurements were taken

## Measurements
The value in the tidy data for each of the variables listed is the MEAN VALUE for all measurements of the given variable recorded by the specified test subject during the specified activity. For example, the tBodyAccMeanX value of 0.2215982 in row 1 of the table is the mean of all tBodyAccMeanX data recorded by subject 1 while laying.

## Activity Labels

* `WALKING`: subject was walking during the test
* `WALKING_UPSTAIRS`: subject was walking up a staircase during the test
* `WALKING_DOWNSTAIRS`: subject was walking down a staircase during the test
* `SITTING`: subject was sitting during the test
* `STANDING`: subject was standing during the test
* `LAYING`: subject was laying down during the test
